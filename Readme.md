# Основные команды в Git

1. git init – инициализация локального репозитория
2. git status – получить информацию от git о его текущем состоянии
3. git add – добавить файл или файлы к следующему коммиту
4. git commit -m “message” – создание коммита.
5. git log – вывод на экран истории всех коммитов с их хеш-кодами
6. git clone <url-адрес репозитория> – клонирование внешнего репозитория на локальный ПК
7. git pull – получение изменений и слияние с локальной версией
8. git push – отправляет локальную версию репозитория на внешний
